# Homebrew Tap for StashFlow

Install StashFlow with Homebrew:

```bash
brew tap iwen-conf/stashflow
brew install stashflow
```

Or install directly:

```bash
brew install iwen-conf/stashflow/stashflow
```

Commands:

- `stashflow`: terminal UI
- `stashflow-clean`: batch CLI

Project: https://github.com/iwen-conf/stashflow
